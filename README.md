# ביוגרפיה של סבא יוסי (קליין)

## המטרה
לכתוב ביוגרפיה איכותית, נוגעת ללב ובעלת נגיעות ספרותיות על חייו של סבא יוסי, המבוססת על שיחות מוקלטות/מתועדות איתו. לא רשימת עובדות יבשה — סיפור חיים עם קול, אווירה, דמויות ורגע.

## איך זה עובד
1. **שיחה** עם סבא (הקלטה או רשימות) → נשמרת ב-`01-ראיונות/`.
2. **תמלול/סיכום** של השיחה → נשמר ב-`02-תמלולים/`, לפי התבנית בקובץ `02-תמלולים/תבנית-תמלול.md`.
3. **כתיבת פרק** על סמך התמלול, בעיבוד ספרותי → נשמר ב-`03-פרקים/`, לפי סדר כרונולוגי או תמטי.
4. **חומרי רקע** (תמונות, מסמכים, שמות, תאריכים, מקומות) → `04-חומרי-רקע/`.
5. כשיש מספיק פרקים בשלים — מרכיבים אותם ל**גרסה הסופית**: קובץ HTML נקי ויפה ב-`05-גרסה-סופית/index.html`.

## מבנה התיקייה
```
01-ראיונות/          תמלילים גולמיים / הקלטות / רשימות משיחה, קובץ per שיחה
02-תמלולים/          תמלול מסודר של כל שיחה (לפי תבנית-תמלול.md)
03-פרקים/            פרקי הביוגרפיה בעיבוד ספרותי, קובץ per פרק
04-חומרי-רקע/
  תמונות/            סריקות/צילומים רלוונטיים
  מסמכים/            תעודות, מכתבים, מסמכים היסטוריים
05-גרסה-סופית/       index.html — הגרסה המורכבת והמעוצבת של הביוגרפיה
מדריך-ראיונות.md      נושאים ושאלות מומלצות לכל שיחה
ציר-זמן.md            ציר זמן חי של אירועים, שמות ותאריכים — מתעדכן תוך כדי
```

## מקורות לכל שיחה
לכל שיחה יכולים להיות כמה סוגי מקור, וכולם רלוונטיים — כדאי לשמור את כולם ב-`01-ראיונות/` (למשל עם קידומת תאריך): טקסט שסבא כתב בעצמו, תרגום (כגון לצרפתית), תמלול/דיבוב של הקלטה, הערות שנרשמו תוך כדי ההאזנה, וסיכום מעורבב. בכתיבת הפרק הסופי יש לצלוב בין כל המקורות הללו כדי לתפוס גם את התוכן וגם את הניסוח המדויק.

## השלמת פערים היסטוריים
מותר ורצוי לחפש ברשת השלמות של הקשר היסטורי/תקופתי (אירועים, מקומות, תנאי חיים) כדי לעגן את הסיפור האישי בזמן ובמקום — ולסמן את זה בפרק כ"הערת רקע היסטורי" נפרדת מהסיפור עצמו (ראו תבנית ב-`03-פרקים/תבנית-פרק.md`).

## שאלות פתוחות
בכל פרק יש לסמן במפורש נקודות שדורשות חידוד עם סבא בשיחה הבאה, תחת "שאלות פתוחות לבירור" — כדי שלא יאבדו בין הפרקים.

## סגנון עיצובי לגרסה הסופית
העיצוב של `05-גרסה-סופית/index.html` מבוסס על סגנון מינימליסטי-אפל (רקע כהה, גופן Heebo לכותרות/ניווט ו-Frank Ruhl Libre לטקסט הרץ, נגיעת צבע כתום-חם), בהשראת אתר החברה של המשפחה. אפשר לעדכן את משתני הצבע בראש קובץ ה-CSS (`:root`) אם יישלח סגנון עדכני יותר.

## עקרונות כתיבה
- **קול אמיתי**: לשמור על הביטויים והניסוחים הייחודיים של סבא, לא "לייפות" אותו לניסוח סטנדרטי.
- **סצנות, לא סיכומים**: איפה שאפשר, להראות רגע קונקרטי (ריח, צליל, משפט שנאמר) במקום לתאר אותו כללית.
- **כרונולוגיה גמישה**: אפשר לארגן לפי תקופות חיים (ילדות, נעורים, צבא, משפחה...) אבל גם לפתוח פרק ב"רגע" ואז לחזור אחורה.
- **דיוק עובדתי**: שמות, תאריכים ומקומות מתועדים ב-`ציר-זמן.md` ונבדקים מול המרואיין כשיש ספק.
- **אורך פרק**: 800–1800 מילים בערך, כל פרק עומד גם בפני עצמו.
